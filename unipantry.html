UniPantry Mobile App

Overview

UniPantry is a startup created by Cornell students. It serves to empower students to live healthy lifestyles, beginning with eating nutritious meals. As students ourselves, we understand the struggle to eat nutritiously while staying within money and time constraints. I am currently working with the product managers to design and launch the first iteration of the mobile app.

Prototyping Iterations

After designing an initial prototype, I made changes to the app based on feedback from my fellow designer, product manager, and users.
Improving the Visual Design of the Grocery List
I designed the initial Grocery List with the vision of a colorful, visually appealing interface. However, upon revisiting my design, I observed that the interface over-highlighted individual grocery items, causing blurred section separation and unclear visual hierarchy. I knew this was a problem because users had indicated that they greatly relied on the clear categorization of items to shop efficiently in grocery stores. Therefore, I made the following UI changes to more clearly represent the users‚Äô mental model:

  • Made the grocery category headers more visible

  • Made the visual separation of individual items more clear

  • Reduced the visibility of ‚ÄúRecently Bought‚Äù items so users can focus their attention on the items they still need

  • Removed the images because they distracted from primary affordances and added a layer of difficulty for the software team

Improving the Usability of the Filtering Function
In order to make the filter function feel more seamless and efficient, I made the following redesign changes:

  • Altered the interaction so that the user can view all their options with one click

  • More clearly communicated to the user that they can choose more than one category option

  • Allowed the user to choose when they want to start applying the filter
